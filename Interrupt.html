Interrupt Testing in Mobile Application using Twilio

Project Information

  • Category: Interruption testing is another level of mobile application testing.
  • Description:Interrupt Testing is a branch of Mobile Application Testing that deals with how an application reacts to interruption and resumes to its previous state.
  • Importance:
    1. While performing an online transaction, the application asks for the One Time Password (OTP). 2. You minimize the application to check the OTP message.
    3. After checking the OTP when you open the application, your transaction fails
  • Technologies: Java, TestNG, Appium, twilio and Maven
  • Project Date: 12 April, 2020
