UI Testing and Monitoring

Project Information

  • Category: Visual Validation
  • Description: After each release user interface will be checked by testers. It is a very headache process. With every realease testers will have to verify the user interfaces of an application manually. This is focusing on solving this problem by automating the process
  • Advantages: Automatically generates the UI differnces and QA teams easily identify colours or alignment issues
  • Technologies: Python, Selenium, TestNG, openCV, Java
  • Supported Platforms: Windows, Linux and Mac OS
  • Project Date: 11 January, 2020
