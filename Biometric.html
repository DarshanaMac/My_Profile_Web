Biometric scenario automation using ADB commands

Project Information

  • Category: Biometric Test | ADB Commands
  • Description: Nowadays most of the mobile application authentication base on the fingerprint. As an example, people are using lot of e-Wallets for their payments. Developers should worry about their security that is why they use fingerprint authentication mechanisms for mobile apps. When we automate regression test suites on mobile, we have to cover fingerprint scenarios as well. Is it possible to automate? What do you think ?? The answer is ............ YES but only works with emulators.
  • Advantages: We can automate frigerprint scenario with emulators and check application and mobile OS works communicate properly this kind of cases
  • Technologies: Java, Appium, ADB Commands
  • Supported Platforms: Android
